Ward Board Site Plan

Noah Merrell

WDD 131-3

Overview

Purpose

To create a website that can list ward events and information dedicated to a ward. Essentially, it'll act as a ward website.

Audience

Members of the ward

Dynamic elements

The site will dynamically change with added ward information and events. This will be done with JavaScript and JSON files which can be added to for more recent information. I intend to take advantage of Make in order to automate the process of creating these JSON files. A searchbar will be added to look up events and the user will be able to filter them based on the type (Elder's Quorum, Relief Society, Ward, etc.). Additionally, other small features such as a light/dark mode will be added to the site.
